Hotjar connector that avoids connections when logged in wp-admin.

== Description ==

Plugin to connect your site with hotjar changing the Hotjar ID from wp-admin.
Also it does not connect with Hotjar while you are logged.

= Get involved =

Developers can contribute to the source code on the [GitHub Repository](https://github.com/thiagogsr/wp-hotjar).

== Installation ==

= Minimum Requirements =

* WordPress 3.8 or greater

= Instructions =

1. Upload the entire 'wp-hotjar' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
